Simon Says Game

Press Any Key to Start The Game

Game Instructions:

  • Starting the Game

    Press any key to start the game

  • Objective

    Watch the sequence of colors that flashes on the buttons. Your goal is to remember and repeat the sequence.

  • Playing the Game

    Once the sequence is shown, click the buttons in the exact order that was displayed.

  • Level Up

    Each round, a new color is added to the sequence, making it progressively harder to remember.

  • Game Over

    If you click the wrong button, the game ends. Your final score will show your level reached

  • Restarting

    Press any key to start a new game and try to beat your previous score.
